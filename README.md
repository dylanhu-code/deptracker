# Cross-project alignment of dependency updates

This project studies how software projects align when the same dependency update appears across many repositories. The study mines pull requests that touch dependency manifests in five ecosystems, Maven, npm, Cargo, pip, and Go, and classifies pull-request sources as Dependabot, Renovate, other bot, or human before analysing update outcomes and cross-project agreement.
